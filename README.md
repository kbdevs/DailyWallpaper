To apply this create an automation in the shortcuts app to request the url of wherever you host this and set it as the lockscreen/homescreen wallpaper.

You likely will want to adjust the coloring and such of certain days to your life.

?style=colored shows colors
?date=mm-dd-yyy to set a certain date
